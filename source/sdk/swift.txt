Use the {+service+} Swift SDK to write Swift and Objective-C mobile 
applications. Access data stored in the free and open-source Realm Database, 
or work with {+backend+} Functions, MongoDB Data Access, and user 
authentication.

.. seealso::

   Are you looking for information about using Swift with MongoDB in a 
   backend app, command-line utility, or running on macOS or Linux? See 
   the :driver:`MongoDB Swift Driver documentation </swift>`.
