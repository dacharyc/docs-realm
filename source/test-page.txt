======================
Tab Selector Test Page
======================

.. tabs-selector:: realm-languages

Just a test page to see if we can successfully use the language selector
drop-down menu with the Realm SDKs tabs.

.. tabs-realm-languages::

   .. tab::
      :tabid: java

      Some Java examples here.

   .. tab::
      :tabid: cpp

      Some C++ examples here.

   .. tab::
      :tabid: javascript

      Some Javascript example here.

   .. tab::
      :tabid: swift

      Some Swift examples here.

      To add an object to a realm, instantiate it as you would any other
      object and then pass it to :swift-sdk:`Realm.add(_:update:)
      <Structs/Realm.html#/s:10RealmSwift0A0V3add_6updateySo0aB6ObjectC_AC12UpdatePolicyOtF>`
      inside of a write transaction.

   .. tab::
      :tabid: objective-c

      Some Objective-C examples here.

      To add an object to a realm, instantiate it as you would any other
      object and then pass it to :objc-sdk:`-[RLMRealm addObject:]
      <Classes/RLMRealm.html#/c:objc(cs)RLMRealm(im)addObject:>` inside
      of a write transaction.

   .. tab::
      :tabid: dart

      Some Dart examples here.

      To add an object to the database, pass an instance of a Realm object class
      to the database in a write transaction block with
      :flutter-sdk:`Realm.add() <realm/Realm/add.html>`.
